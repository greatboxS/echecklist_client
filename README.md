# echecklist_client
Serial baud: 115200

1. Setting AREA Id:
==> Send: device-setting
==> Send: AreaID, 1 ~ 7

2. Setting Server IP: 
==> Send: setting-ip
==> Send: 10.10.10.3

3. Setting Server Port:
==> Send: setting-port
==> Send: 45520

4. Setting Mac:
==> Send: setting-mac
==> Send: MAC ID, example    
20:13:e2:4e:bb:c1
